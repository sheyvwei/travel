分支测试
远程创建分支dev
本地同步分支git pull
本地切换分支git checkout dev
提交代码到dev
git add .
git commit -m 
git push
合并dev到master
1先切换到master: git checkout master
2.合并到master git merge dev
3.把master的内容也提交到远程仓库 git push
4.删除本地dev分支 git branch -d dev
5.删除远程分支 git push :dev
6.查看所有分支（包括远程分支) git branch -a
